# Survey Data Analysis: User Behaviour in Academic Digital Libraries

## Project Overview
This project analyzes survey data collected as part of a Master's dissertation research study on understanding user behaviour in academic digital libraries and the role of big data analytics in improving digital library services.

The objective of the analysis is to explore patterns in how academic professionals interact with digital library platforms and their awareness of data analytics tools.

## Dataset
## Note on Dataset

The original survey dataset used in the MSc dissertation is not publicly available due to research data handling practices.

To demonstrate the analysis workflow, a **synthetic sample dataset** with similar structure and variables has been created for this repository.

The dataset replicates the types of responses collected in the original survey while not containing any real respondent data.

## Analysis Performed

The analysis includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Frequency distribution analysis
- Visualization of survey responses
- Identification of trends in digital library usage

## Key Questions Explored

1. What devices are commonly used to access digital libraries?
2. What is the level of awareness of big data analytics among librarians?
3. What skills are considered important for managing digital library data?
4. What training methods are preferred for developing analytics skills?

## Key Insights

Some insights from the analysis include:

- Smartphones were the most common device used to access digital library platforms.
- Awareness of big data analytics among respondents was moderate but increasing.
- Data analysis and data visualization were identified as key skills needed for managing digital library services.
- Practical training and online courses were the most preferred learning methods.
